# Izkor Data Extractor

סקריפט Python לחילוץ מידע מאתר יזכור (izkor.gov.il) על חללים לפי בתי עלמין, ושמירת הנתונים לקובץ Excel.

הסקריפט משתמש בבקשות HTTP ל-API של האתר, מסנן בתי עלמין לפי רשימה מותאמת אישית, אוסף את הנתונים של החללים מכל עמודי התוצאות (pagination), ומייצר קובץ Excel עם כל המידע.

---

# מה הסקריפט עושה

הסקריפט מבצע את השלבים הבאים:

1. שולח בקשה ל-API של האתר כדי לקבל רשימת בתי עלמין.
2. מסנן את הרשימה לפי שמות בתי עלמין שמוגדרים בקובץ `filter_cities.txt`.
3. עבור כל בית עלמין שנמצא:

   * שולח בקשות ל-API שמחזירות את רשימת החללים הקבורים בו.
   * עובר על כל עמודי התוצאות.
4. מוסיף לכל חלל את שם בית העלמין ממנו נשלף.
5. מאחד את כל הנתונים.
6. שומר את הכל לקובץ Excel בשם:

```
izkor_data.xlsx
```

# חילוץ Cookies מהדפדפן

האתר משתמש בהגנה נגד בוטים, ולכן יש צורך להשתמש ב-cookies מהדפדפן.

## שלבים

1. פתח את האתר:

```
https://www.izkor.gov.il
```

2. לחץ `F12` כדי לפתוח **Developer Tools**

3. עבור ללשונית:

```
Network
```

4. סנן לפי בקשות:

```
Fetch / XHR
```

5. רענן את הדף.

6. מצא את הבקשה:

```
GET https://www.izkor.gov.il/search/list-fields
```

7. לחץ עליה.

8. עבור ללשונית:

```
Headers
```

9. העתק את הערך של:

```
Cookie
```

לדוגמה:

```
visid_incap_XXXXX=...; nlbi_XXXXX=...; incap_ses_XXXXX=...
```

---

# עריכת ה-Cookies בסקריפט

פתח את קובץ הסקריפט ועדכן את ה-headers:

```python
headers = {
    "User-Agent": "Mozilla/5.0",
    "Accept": "application/json",
    "Cookie": "PASTE_YOUR_COOKIE_HERE"
}
```

יש להדביק את ה-cookie שהועתק מהדפדפן במקום המתאים.

# הגדרת בתי העלמין לסינון

הסקריפט משתמש בקובץ:

```
filter_cities.txt
```

הקובץ צריך להכיל רשימה של שמות בתי עלמין בעברית.

דוגמה:

```
בית העלמין הצבאי ראשון לציון
בית העלמין הצבאי אשדוד
בית העלמין הצבאי באר שבע
ירושלים
```

כל שורה בקובץ היא מחרוזת חיפוש.

הסקריפט יחלץ רק בתי עלמין ששמם **מכיל אחת מהמחרוזות** בקובץ.

# פלט הסקריפט

בסיום הריצה ייווצר קובץ:

```
izkor_data.xlsx
```

הקובץ יכיל את כל החללים שנמצאו, עם שדות כמו:

* full_name
* first_name
* last_name
* rank
* death_date
* age
* father_name
* mother_name
* service_type
* cemetery_name

כל חלל יופיע בשורה נפרדת.
